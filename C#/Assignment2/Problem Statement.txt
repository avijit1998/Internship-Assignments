The task requires us to build a console application where the user will be asked to input his/her income and 
the amount of the money he/she has invested(under 80C). In the output the following will be displayed,
1>The Taxable Income after deductions (if any)
3>The amount of tax in each slabs(max. 3) the user must pay based on his/her salary. 

Additional Requirements
The solution must contain three projects
1.TaxCalculator.UI (only to be used for I/O)
2.TaxCalculator.Business (Library Class containing business logic implementing the interfaces)
3.TaxCalculator.Infra (Containing the interfaces for all business logic)
